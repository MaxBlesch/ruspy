ruspy.data package
==================

Introduction
-----------
In this package of the ruspy project are modules to read and process the original data provided by John Rust. The package is split into three modules,
each of them described below. The readme of the raw data can be found [here](https://github.com/OpenSourceEconomics/ruspy/blob/master/ruspy/data/original_data/README).
The documentation on the structure of the data, which Rust used and my data has after the data reading process in the data_reading module below can be found
[here](https://github.com/OpenSourceEconomics/ruspy/blob/master/documentatio/nfxp_docu.pdf).
